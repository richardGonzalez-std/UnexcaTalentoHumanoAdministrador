# UnexcaTalentoHumanoAdministrador

Sistema de gestión para el departamento de Talento Humano de la UNEXCA.

## Objetivo Principal
Automatizar procesos administrativos internos, con especial énfasis en el cálculo de prestaciones sociales para reducir la dependencia de servicios externos.

## Módulos del Sistema
1. **Login:** Control de acceso seguro para el personal autorizado.
2. **Cálculo de Prestaciones:** Automatización de cálculos según normativas legales (Ministerio de Educación).
3. **Administración:** Gestión general de datos y configuraciones del sistema.
4. **Correspondencia:** Manejo y seguimiento de documentos y comunicaciones oficiales.

## Tecnologías
### Backend
- Java 26
- Spring Boot 4.0.6
- MySQL
- Spring Modulith

### Frontend
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS

## Estructura del Proyecto
- `/back`: Aplicación Spring Boot (API REST).
- `/front`: Aplicación Next.js (Interfaz de usuario).
